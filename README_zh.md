## 🚀 快速开始

### 系统要求

- **Python 3.11+** (推荐 3.12+)
- **Pygame 2.2+**
- **macOS / Linux / Windows**

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/lulin70/PyCC2.git
cd PyCC2

# 创建虚拟环境
python -m venv .venv
source .venv/bin/activate   # macOS/Linux
# .venv\Scripts\activate   # Windows

# 安装包（开发模式）
pip install -e .

# 或安装含开发依赖的完整版本
pip install -e ".[dev]"
```

### 启动游戏

```bash
# 启动游戏
pycc2

# 或使用 Python 模块方式
python -m pycc2.main
```

### 操作说明

| 操作 | 输入方式 |
|------|----------|
| 选择单位 | 鼠标左键点击 |
| 下达命令 | 鼠标右键拖拽（径向菜单）或热键 (Z/X/S/C/V/D/H) |
| 多选单位 | Shift + 左键 |
| 队列命令 | Shift + 右键 |
| 平移摄像机 | 方向键 / WASD / 边缘滚动 |
| 缩放视角 | 鼠标滚轮 |
| 暂停 | ESC（菜单）/ Space（时间控制） |
| 视线检查 | 按住 Ctrl |

## 🏗️ 技术架构

```
PyCC2/
├── src/pycc2/
│   ├── domain/              # 核心游戏逻辑（纯Python，高度可测试）
│   │   ├── ai/             # 行为树AI、战术决策系统
│   │   ├── components/     # ECS组件：血量、士气、武器、位置、视野
│   │   ├── entities/       # 班组、单位、游戏地图、抛射物
│   │   ├── systems/        # 战役、战斗、弹道、路径规划
│   │   └── value_objects/  # 伤害、方向、地形类型、向量
│   ├── services/           # 游戏循环、AI服务、事件总线、战斗导演
│   ├── presentation/       # 渲染、输入处理、UI、音频
│   │   ├── rendering/      # 摄像机、HUD、小地图、精灵、等距引擎
│   │   │   ├── surface_pool.py  # 统一 Surface LRU 池
│   │   │   └── fade_transition.py  # UI alpha淡入淡出过渡辅助类
│   │   ├── input/          # 命令系统、交互控制器
│   │   ├── ui/             # 菜单、面板、提示框、部署UI
│   │   └── audio/          # 音效系统、语音命令
│   └── infrastructure/     # 存档系统、配置、解析器
│       └── resource_cache.py        # HTTP资源下载与本地缓存管理
├── data/
│   ├── maps/               # 63张历史地图JSON文件
│   ├── scenarios/          # 11个场景配置
│   └── units/              # 单位模板定义
├── tests/                  # 4573个测试（单元+集成+E2E+烟雾测试）
├── assets/                 # 精灵、音效、CC2参考截图
└── docs/                   # 设计文档、PRD、差距分析
```

**设计原则**：
- **领域驱动设计 (DDD)**: 业务逻辑与基础设施清晰分离
- **事件驱动架构**: EventBus 实现系统间松耦合
- **固定时间步长**: 逻辑 @30 UPS, 渲染 @60 FPS
- **基于组件的实体 (ECS)**: 灵活的单位组合模式
- **行为树AI**: 模块化、可扩展的AI决策框架
