## はじめに

**PyCC2**へようこそ。1944年9月17日〜25日の**マーケット・ガーデン作戦**を舞台にした戦術的第二次世界大戦戦闘シミュレーターです。オランダの田園地帯で枢軸軍との過酷な接近戦において連合軍歩兵中隊を指揮します。

これは**アケードゲームではありません**。パワーアップも、リスポーンも、回復アイテムもありません。あなたの部隊は限られた弾薬、脆い士気、そして死すべき兵士を持っています。すべての決定が重要です。

**あなたの役割**: 中隊指揮官です。あなたはトリガーを引くのではありません — あなたの兵士たちが引きます。あなたは命令を出します：*あそこへ移動せよ。あれを撃て。ここで掩体を取れ。*あなたの仕事は、適切な部隊を適切な場所に適切なタイミングで配置することです。

## 開始方法

### ゲームの起動

```bash
# ゲームを開始
pycc2

# またはPythonモジュールとして実行
python -m pycc2.main
```

### メイン画面

起動すると、以下が表示されます：

```
+----------------------------------------------------------+
| [FPS:60] Tick:0150  Time:00:50  Turn:1  [PAUSED]        |  <- トップバー (HUD)
+----------------------------------------------------------+
|                                                          |
|    +-----+                                                |
|    | o * |  <- 選択されたユニット（黄色いリング）         |
|    +-----+  Health: XXXX..  Morale: XXXXXX                |
|                                                          |
|          # # #                                           |  <- 敵ユニット（灰色）
|                                                          |
|    ........                                              |  <- 森林（濃い緑）
|    ........                                              |
|                                                          |
|  =======                                                 |  <- 道（灰色）
|                                                          |
|  [Minimap]                     [Command Bar]             |  <- ボトムHUD
|  XXXXXXX___    [Move][Attack][Hold][Dig][X]              |
+----------------------------------------------------------+
```

### 基本的なフロー

1. **選択**: 左クリックでユニットを選択
2. **命令**: 右クリック（地面 = 移動、敵 = 攻撃）
3. **監視**: HPバー、士気、弾薬を確認
4. **勝利**: 敵司令官を排除するか全敵軍を破壊して勝利

## ユニットガイド

### 連合軍（プレイヤー）

デモはマップ左側に**7つの連合軍ユニット**から始まります。

#### 歩兵分隊（ライフル分隊）

| ステータス | 値 |
|-----------|-----|
| HP | 100 |
| 主武器 | ライフル（10発） |
| 発射ごとのダメージ | 8 – 18 |
| 視界範囲 | 5タイル |
| 移動速度 | 3.0 タイル/秒 |
| 基礎士気 | 85 |
| 役割 | 汎用作業馬 |

**最適用途**: 位置の維持、火の下での前進、側面攻撃機動。あなたの主力戦力。バランスの取れたステータスによりあらゆる状況で万能。

#### 機関銃分隊（MG班）

| ステータス | 值 |
|-----------|-----|
| HP | 80 |
| 主武器 | MG42（50発） |
| 発射ごとのダメージ | 12 – 25 |
| 視界範囲 | 6タイル |
| 移動速度 | 2.0 タイル/秒 |
| 基礎士気 | 75 |
| 役割 | エリア制圧、防衛の要 |

**最適用途**: 防衛位置、要路の確保、援護射撃の提供。移動が遅いため移動時に脆弱。大型マガジン（50発）は持続的な火力能力を意味します。

#### スナイパーチーム

| ステータス | 値 |
|-----------|-----|
| HP | 60 |
| 主武器 | スナイパーライフル（15発） |
| 発射ごとのダメージ | 25 – 50 |
| 視界範囲 | **10タイル** |
| 移動速度 | 2.5 タイル/秒 |
| 基礎士気 | 80 |
| ステルスボーナス | **+40% 隠蔽** |
| 役割 | 長距離精密攻撃 |

**最適用途**: 距離から露出したターゲットを排除し、拡張視界で敵を発見。掩体の後ろに置いてください — 60HPしかなく非常に脆弱！1発あたりのダメージは高いが弾薬が限られるため精度が必要。

#### 衛生兵チーム

| ステータス | 値 |
|-----------|-----|
| HP | 70 |
| 主武器 | ピストル（12発） |
| 発射ごとのダメージ | 4 – 8 |
| 視界範囲 | 5タイル |
| 移動速度 | 3.0 タイル/秒 |
| 治療範囲 | 3タイル |
| 治療レート | 0.5 HP/tick |
| 基礎士気 | 88 |
| 役割 | サポート — 近くの味方を治療 |

**最適用途**: 前線分隊の戦闘継続時間を延長。戦闘の密集地の近く（しかし中にはない）に配置。白いヘルメットスプライトは戦場での目印となります。

#### 司令官（Cpt. Miller）

| ステータス | 値 |
|-----------|-----|
| HP | 100 |
| 主武器 | ピストル（14発） |
| 発射ごとのダメージ | 6 – 12 |
| 視界範囲 | **7タイル** |
| 移動速度 | 3.0 タイル/秒 |
| 基礎士気 | **95** |
| 役割 | **重要資産 — 喪失 = 敗北の可能性** |

**最適用途**: 後方地域の調整（7タイルの高視界範囲）。**絶対に守ってください** — 司令官を失うことは即時の敗北条件となることが多く、すべての味方ユニットに莫大な士気ペナルティを与えます。

### 枢軸軍（AI敵）

AIはデモで**7つの同等のユニット**を制御します。以下と対戦することになります：

| ユニット | 名前（ゲーム内） | 脅威レベル |
|---------|-----------------|-----------|
| x3 歩兵分隊 | Grenadier-1, Grenadier-2, Grenadier-3 | 中 — 主攻撃力 |
| x1 MG班 | MG班 | 中 — 防衛の要 |
| x1 司令官 | **Oberst Krebs** | **重要 — あなたの主目標** |
| x1 戦車 | Panzer IV | **高 — 主要な脅威** (200 HP, 35-70 ダメージ) |
| x1 スナイパー | Scharfschutze | 高 — あなたの露出ユニットを狙う |

> [!IMPORTANT]
> **枢軸軍側は完全にAI制御**されており、CommanderAI、SquadCoordinator、ビヘイビアツリー意思決定、難易度スケーリングを備えています。あなたは本物の対戦相手と戦っています。

## 地形ガイド

戦場には**14種類の地形**があり、それぞれ移動、戦闘、視認性に影響を与える独自の特性があります。

### 移動コスト（低いほど高速移動）

| 地形 | 移動コスト | 掩体ボーナス | 隠蔽 | 通過可能? | LOS遮断? |
|------|-----------|-------------|------|----------|----------|
| **開地** | 1.0x | 0% | 0% | はい | いいえ |
| **道路** | 0.8x | 0% | 0% | はい | いいえ |
| **草地** | 1.2x | 5% | 15% | はい | いいえ |
| **森林** | 2.0x | **20%** | **50%** | はい | **はい** |
| **建物（進入可能）** | 1.5x | **50%** | **70%** | はい | いいえ |
| **建物（固体）** | 無限 | **80%** | **90%** | いいえ | **はい** |
| **水域** | 無限 | 0% | 5% | いいえ | **はい** |
| **生け垣** | 2.5x | 15% | 35% | はい | いいえ |
| **壁** | 無限 | **70%** | **80%** | いいえ | **はい** |
| **荒地** | 1.8x | 8% | 25% | はい | いいえ |
| **浅瀬** | 3.0x | 5% | 10% | はい | いいえ |
| **橋** | 0.9x | 0% | 0% | はい | いいえ |
| **弾痕** | 2.5x | **25%** | 20% | はい | いいえ |
| **沼地** | 4.0x | 0% | 30% | はい | いいえ |

### 視線（LOS）

これらの地形は視線を**完全に遮断**します — 通して射击することはできません：

- **森林** — 密な森は視界を遮断；待ち伏せ位置として使用
- **固体建物** — 完全遮断; 近くに重い掩体を提供
- **壁** — 完全遮断; 隣接タイル用の硬い掩体
- **水域** — 完全遮断; そもそも通行不可

**戦術的使用**: あなたのスナイパーまたはMG班を森林または固体建物の後ろに隠してください。敵が開地に入るのを待ち、安全な位置から交戦してください。

## 戦闘メカニクス

### 射撃プロセス

あなたの部隊が敵を攻撃すると、ゲームはこの解決パイプラインを実行します：

```
1. 射程チェック
   -> ターゲットが武器射程内か？（ユニットタイプにより異なる）
   -> 最大有効射程は武器により異なる（スナイパー > ライフル > ピストル）

2. 視線（LOS）
   -> パスは clear か？（タイルグリッドを通るBresenham raycast）
   -> 以下により遮断：森林、固体建物、壁、水域
   -> 地形の高低差による部分的影響

3. 弾道計算（BallisticEngine）
   -> 命中確率は以下に基づく：
      * ターゲットまでの距離（近いほど正確）
      * ターゲットの掩体ボーナス（建物内 = かなり命中しにくい）
      * 射手の経験（難易度設定修正）
      * 予測不能性のための乱数要素

4. ダメージロール
   -> 命中した場合：ダメージ = random(武器最小, 武器最大)
      距離と掩体により修正
   -> 典型的な結果：
      * ピストル命中: 4-12 HP
      * ライフル命中: 8-18 HP
      * MG42命中: 12-25 HP
      * スナイパー命中: 25-50 HP
      * 戦車砲: 35-70 HP
      * 迫撃砲: 20-45 HP
      * 対戦車砲: 30-60 HP

5. 士気インパクト
   -> 両軍の士気は以下に影響される：
      * 被害を受ける（大きな士気低下）
      * 近くで味方が死亡（パニック連鎖のリスク）
      * 持続的な銃撃下にある（制圧減衰）
```

### 掩体は重要 — 非常に

歩兵が**開地**にいる場合（掩体ボーナス0%）、50-60%の確率で命中するかもしれません。
同じ歩兵が**進入可能な建物**内にいる場合（掩体ボーナス50%）、10-20%の確率でしか命中しなくなります。
**固体建物**の後ろ（掩体ボーナス80%）では、その方向からの命中率はほぼゼロに低下します。

**常に掩体のある位置に接近してください。可能な限り掩体から戦ってください。**

### 士気システム

各ユニットには戦闘有効性を支配する士気値（0–100）があります：

| 士気範囲 | 状態 | 戦闘効果 |
|----------|------|----------|
| 86 – 100 | **自信あり** | 完全な戦闘有効性、通常の精度 |
| 51 – 85 | **正常** | 標準的なパフォーマンス、わずかな精度変動 |
| 31 – 50 | **動揺** | 精度低下、命令のためらい |
| 11 – 30 | **制圧** | 命令を拒否する可能性あり、釘付け、非常に低い精度 |
| 0 – 10 | **崩壊** | 敗走 — ユニットは逃走または降伏し、事実上戦闘不能 |

**士気が低下するとき**:
- 分隊員が死亡する（特に近くのユニット）— **大きなペナルティ**
- 短時間に重度の被害を受ける — 累積効果
- 司令官が死亡する — **軍全体への莫大なペナルティ**
- 圧倒的に数で劣る — 漸進的な圧力

**士気がゆっくり回復するとき**:
- 数tick間被害を受けないとき
- 味方ユニットの近く（近接ボーナス）
- 司令官が生存し視界内にいる时候

## AI対戦相手

### 敵司令官が何を考えているか

枢軸軍AIは**4つの異なる層**を持つ多層決定システムを使用します：

#### 層1: 知覚システム

各AIユニットは戦場について独自の知覚を維持します：
- この特定のユニットは何を見ることができるか？（フォッグ・オブ・ウォーを尊重 — AIはチートしません！）
- 検出された敵はどこにいるか？
- 各検出ターゲットの脅威レベルは？
- ステルスボーナスが適用（スナイパーは検出されにくい）

#### 層2: 部隊コーディネーター

部隊がチームとしてどのように協力するかを管理：
- **火力集中**: 複数の部隊が1つの高価値ターゲットに火力を集中
- **交差支援**: 1つの部隊が移動している間、もう1つが固定位置からカバー
- **交差火線セットアップ**: 部隊を2以上の角度から同時に攻撃できるように配置
- **側面機動**: 主力が敵の注意を固定している間、側面からユニットを送る

デモでは、2つの分隊が登録されています：
- `axis_alpha`: 3つの歩兵ユニット（調整された攻撃グループ）
- `axis_bravo`: MG班 + 司令官（サポート/指揮グループ）

#### 層3: 司令官AI（Oberst Krebs）

敵司令官による戦略レベルの評価：
- 全体的な**戦力比率**を評価（自分の方が強いか弱いか？）
- 5つの階層で**脅威レベル**を評価: なし → 低 → 中 → 高 → 致命的
- 完全な戦場の絵に基づく戦略的命令を生成
- 戦闘の進行に基づきスタンスを調整

#### 層4: ユニットごとのビヘイビアツリー

各個別のAIユニットは戦術的決定のため独自の**ビヘイビアツリー（BT）**を実行します：

```
Root
+-- Sequence: Engage Target
|   +-- Selector: Have Target?
|   |   +-- Condition: Target Assigned?
|   |   +-- Action: Acquire Nearest Enemy
|   +-- Sequence: Attack
|       +-- Condition: In Range?
|       +-- Condition: Has Ammo?
|       +-- Condition: LOS Clear?
|       +-- Action: Fire Weapon
|       +-- Action: Reload if Empty
+-- Sequence: Move Tactics
|   +-- Selector: Decision
|       +-- Condition: Health Low? -> Retreat
|       +-- Condition: Enemy Close? -> Find Cover
|       +-- Default -> Advance / Patrol
+-- Fallback: Idle / Patrol
```

異なるユニットタイプは特殊化されたBTファクトリを使用します（歩兵BT、MG分隊BT、司令官BTなど）。

### 難易度レベル

| レベル | 名前 | AI知覚 | AI命中率 | AI弾薬 | AI士気 |
|-------|------|--------|----------|---------|--------|
| 1 | **新兵** | 範囲縮小 | -20%ペナルティ | 限定供給 | 壊れやすい（低閾値） |
| 2 | **通常** | 通常 | ベースライン | 通常供給 | 標準的挙動 |
| 3 | **ベテラン** | 範囲拡大 | +15%ボーナス | 豊富な供給 | 圧力下で安定 |
| 4 | **英雄** | 完全視界 | +30%ボーナス | 実質無限 | 不屈の士気 |

デフォルトデモは**MEDIUM（通常）**難易度を使用します。18個のパラメータが難易度レベルごとに調整可能です。

## 勝利と敗北

### 勝利条件（いずれか1つで勝利達成）

| 条件 | 説明 |
|------|------|
| **敵司令官の排除** | Oberst Krebsを殺す — 最速の勝利ルート |
| **全敵の破壊** | すべての枢軸軍ユニットが死亡または敗走（士気 <= 10） |
| **士気崩壊** | 平均枢軸軍士気が10以下に低下（彼らは降伏） |
| **目標の確保** | 特定の位置を所要時間保持 |
| **時間勝利** | タイマー満了時の最も多くの生存ユニット（消耗戦勝利） |

### 敗北条件（いずれか1つで敗北発生）

| 条件 | 説明 |
|------|------|
| **司令官の戦死** | Cpt. Millerが死亡 — 通常即時の敗北 |
| **全軍の損失** | すべての連合軍ユニットが死亡または敗走 |
| **士気崩壊** | 平均連合軍士気が10以下に低下 |
| **時間敗北** | タイマー満了時の敵より少ない生存ユニット |

### 戦後画面

戦闘終了時（勝利または敗北）：
- ゲームが自動的に一時停止
- 半透明のオーバーレイが表示
- 統計パネルに表示：
  - 戦闘時間（ゲームtick / リアルタイム）
  - キルと損失（両軍）
  - 命中率パーセンテージ
  - キルレシオ
  - 与ダメージ/被ダメージ
- **ESC**または**R**を押して戻る/終了

## 戦略ガイド

### 初心者のヒント（まずこれを読んでください）

1. **司令官を守ってください** — Cpt. Millerは最も価値のある資産です（基礎士気95、7タイル視界）。前線から2-3タイル後ろに置いてください。彼の死亡 = 即時の敗北の可能性。

2. **掩体を積極的に使用してください** — 森林（20%掩体）または進入可能な建物（50%掩体）で戦えるなら、開地で戦ってはいけません。弾道エンジンは有意な命中率減少を適用します。

3. **火力を集中させてください** — 1つの敵を完全に破壊する方が、3つを軽く傷つけるよりも良いです。死んだ敵は撃ち返しません。

4. **側面を露出させないでください** — AIのSquadCoordinatorはあなたの側面を狙おうとします。予備分隊を側面、特に司令官の側面監視に残しておいてください。

5. **弾薬に注意してください** — 間違ったタイミングでのリロードは部隊を脆弱にします。マガジン数を追跡してください：ライフルは10発しかありません。

6. **MG班は防衛の要です** — Bravo-MG（80 HP、50発マガジン、12-25ダメージ）は固定位置で壊滅的ですが、痛いほど遅い（2.0 タイル/秒）。早めに設置し、頻繁に再配置しないでください。

7. **スナイパーには忍耐が必要です** — Hawkeye（60 HP、10タイル視界、25-50ダメージ、+40%ステルス）はガラスキャノンです。最大射程とダメージ、最小の生存性。常に森林または建物に置いてください。

### 中級戦術

8. **交差支援** — 1つの歩兵分隊を前進させながら、MG班が固定位置からカバーします。最初の分隊が掩体に到着したら、彼らがカバーしている間に次の分隊が移動します。遅いですが露出を最小化。

9. **「金槌と金床」** — MG班で敵を固定（金床 — 持続的な制圧射撃）、次に歩兵分隊で側面から攻撃（金槌 — 側面からの決定的突撃）。

10. **士気管理** — どの分隊も孤立させないでください。近くに味方のいないまま被害を受け続ける単独分隊は、はるかに早く崩壊します。可能な限りユニットを3-4タイル以内に保ってください。

11. **地形の回廊を使用してください** — 道路は最快の移動（0.8xコスト）を提供しますがゼロの保護です。森林は優れた掩体（20%）と隠蔽（50%）を提供しますが速度を半減（2.0xコスト）します。各状況に戦術的バランスを見つけてください。

12. **戦車の扱い** — 敵Panzer IV（200 HP、1発35-70ダメージ）は主要な脅威です。そのサイズのためすべての敵火を引き付けます。戦車を持っている場合は、歩兵でサポートしてください。敵戦車に対しては：スナイパー + MG + 集中的ライフル射撃を調整してください。

13. **衛生兵の配置** — Doc（3タイル内で0.5 HP/tick治療）はメインラインから2タイル後ろに留める必要があります。これにより前線ユニットへの最大カバレッジが得られ、直接の射線から外れます。

### 上級コンセプト

15. **時間的優位性** — AIはゲームtickごと（30 UPSで約33ms）に決定を処理します。より速い戦術的決定 = より多くのアクションが実行されます。長く考えすぎて一時停止しないでください — モメンタムは重要です。

16. **制圧チェーン** — 殺さなくても、敵を制圧（士気 < 30）することで事実上戦闘から除外できます。彼らは命令を拒否し、射撃を止め、甚至逃走するかもしれません。MG班は高発射率と50発マガジンによりエリア制圧に優れています。

17. **司令官スナイプ** — 最速の勝利条件はOberst Krebsを殺すことです。リスキーですが保護されていますが、効率的：1つの well-placed なスナイパーショットまたは集中的な一斉射撃で戦闘を瞬時に終わらせることができます。彼が露出しているのを目撃したら試す価値があります。

19. **AI挙動の活用** — AIのビヘイビアツリーは最近の脅威を優先し、予測可能なパターン（巡回ポイント、接触への反応）に従います。これらのパターンを学び、予測可能な接近ルートで待ち伏せを設定してください。

20. **勝利条件の選択** — いつも全員を殺す必要はありません。敵の士気が崩壊している場合（複数のユニットが士気30未満）、最後の兵士まで狩るよりも、質量降伏を誘発するために強く押してください。

## キャンペーンミッション

PyCC2には難易度が漸増する**5つの事前定義ミッション**を持つ**キャンペーンシステム**が含まれています：

### ミッション1: 初遭遇（チュートリアル難易度）

- **目的**: 枢軸軍偵察パトロールを排除
- **あなたの戦力**: 3歩兵 + 1司令官
- **敵戦力**: 3歩兵 + 1司令官
- **マップ**: チュートリアル（20x20、基本的な地形ミックス）
- **ヒント**: 操作とメカニクスへの直感的な入門。選択、移動、攻撃命令の学習に集中してください。小さいマップ意味着より速い交戦。このミッションには**対話型チュートリアルプロンプト**が含まれ、新規プレイヤーを基本案内します。

### ミッション2: 橋が遠すぎる（通常難易度）

- **目的**: 時間制限内に橋梁渡河を確保
- **あなたの戦力**: 3歩兵 + 1MG + 1スナイパー + 1司令官
- **敵戦力**: 3歩兵 + 1MG + **1戦車** + 1司令官
- **マップ**: 橋梁強襲（24x24、橋タイル、弹痕、沼地セクターを含む）
- **ヒント**: Panzer IVがすべてを変えます。スナイパー（Hawkeye）を使用して最大射程から弱体化させ、MG援護射撃を調整し、時間切れになる前に目標に急いでください。沼地セクターは移動を遅くします — 避けるか障壁として使用してください。

### ミッション3: 防衛線を維持（ベテラン難易度）

- **目的**: 防衛的立ちは — 時間制限位置を保持
- **あなたの戦力**: 3歩兵 + 1MG + 1衛生兵 + 1迫撃砲 + 1司令官
- **敵戦力**: 3歩兵 + 1MG + 1戦車 + 1スナイパー + 1司令官
- **マップ**: 防衛線（より大きいマップ、複数の接近回廊を持つ多様な地形）
- **ヒント**: 防衛は攻撃より難しい。衛生兵（Doc）を使用して分隊の戦闘継続時間を延長してください。迫撃砲班は密集した敵に対する間接砲撃を提供します。彼らを1点突破させないでください — 縦深的防衛。

### ミッション4: 反撃（ベテラン+難易度）

- **目的**: 突破および敵増援の排除
- **あなたの戦力**: 4歩兵 + 1MG + 1スナイパー + 1衛生兵 + 1迫撃砲 + 1司令官
- **敵戦力**: 4歩兵 + 2MG + **1戦車** + 1スナイパー + 1対戦車砲 + 1司令官
- **マップ: | 開野（限られた掩体を持つ大きな開放地形）
- **ヒント**: 敵の対戦車砲はあなたの戦力に対する主要な脅威です。煙（利用可能な場合）または地形マスキングを使用して接近してください。複合兵器を調整：歩兵が火を引きつけている間、側面部隊が弱点を攻撃。衛生兵が持続作戦で重要になります。

### ミッション5: 最終突撃（英雄難易度）

- **目的**: 完全な殲滅または司令部の占領
- **あなたの戦力**: 完全中隊 — 5歩兵 + 2MG + 1スナイパー + 1衛生兵 + 1迫撃砲 + 1戦車 + 1司令官
- **敵戦力 | エリート部隊 — 5歩兵 + 2MG + **2戦車** + 1スナイパー + 1衛生兵 + 1対戦車砲 + 1迫撃砲 + 1司令官（Oberst Krebs）
- **マップ**: 司令部（複雑な都市/地形ミックス、複数の勝利ポイントを持つ）
- **ヒント**: これはあなたの指揮能力の究極のテストです。今や装甲サポートがあります — 賢く使用してくださいが、敵の対戦車砲から保護してください。敵司令官（Oberst Krebs）は重警護されています；暗殺には正確な調整が必要です。すべてのユニットが重要です — 最終突撃のために戦力を節約してください。

> [!NOTE]
> ゲームは `pycc2` または `python -m pycc2.main` で起動し、フルキャンペーンシステムにアクセスできます。ミッション固有の構成については、`src/pycc2/domain/systems/campaign.py`のキャンペーンシステムソースコードをご覧ください。

## チュートリアルシステム

PyCC2には、新規プレイヤーがゲームメカニクスを段階的に学べるよう設計された**統合チュートリアルシステム**が含まれています：

### チュートリアル機能

- **対話型プロンプト**: ゲームプレイ中に文脈に敏感なヒントが表示
- **進行的レッスン**: 習熟を実証すると新しい概念が解放
- **プラクティスモード**: 結果なしで実験できる安全な環境
- **クイックリファレンス**: ゲームプレイ中に**F1**キーでいつでもアクセス可能

### チュートリアルレッスン

| レッスン | トピック | カバーされるメカニクス |
|---------|----------|---------------------|
| 1 | 基本選択 | 左クリック、Shiftでの複数選択 |
| 2 | 移動 | 右クリック（地面）、経路探索 |
| 3 | 戦闘基本 | 攻撃命令、射程インジケータ |
| 4 | 掩体システム | 地形ボーナス、配置 |
| 5 | 士気管理 | 士気バー、パニック防止 |
| 6 | 上級戦術 | 側面攻撃、制圧射撃 |
| 7 | ユニット特化 | 各ユニットタイプの独自の役割 |

### チュートリアルへのアクセス

- ゲームプレイ中いつでも**F1**を押してチュートリアルオーバーレイを切り替え
- 最初の起動で自動的にチュートリアル（ミッション1）を開始
- メインメニューまたはキャンペーン画面から再プレイ可能
- 進捗はキャンペーンセーブとは独立に保存されます

## セーブシステム

PyCC2は**HMAC-SHA256暗号整合性検証**付きの**8つのセーブスロット**をサポートしています：

### クイックセーブ/ロード

| キー | アクション |
|-----|-----------|
| **F5** | スロット0にクイックセーブ（以前のクイックセーブを上書き） |
| **F9** | スロット0からクイックロード（最新のクイックセーブ） |

### セーブファイルの場所

```
PyCC2/saves/
+-- save_slot_0.json    <- クイックセーブスロット（F5/F9）
+-- save_slot_1.json
+-- save_slot_2.json
+-- ...
+-- save_slot_7.json
```

### セーブファイル整合性（改ざん防止）

各セーブファイルにはセーブデータ全体で計算された**HMAC-SHA256暗号署名**が含まれています。ロード時：

1. ゲームがセーブJSONデータを読み込み
2. 秘密鍵を使用してHMAC-SHA256を再計算
3. 保存された署名 vs 再計算された署名を比較
4. **不一致の場合**: セーブが**破損**としてマーク — ロードを拒否
5. **バージョン間で形式が変更された場合**: セーブが**互換性なし**としてマーク

これにより防止されます：
- HP値の無敵への編集
- 弾薬数の変更
- ユニット位置の変更
- セーブファイルチートのあらゆる形態

## トラブルシューティング

### ゲームウィンドウが表示されない

- pygameがディスプレイにアクセスできることを確認（INSTALL_ja.mdのプラットフォーム固有の注意を参照）
- 環境変数の設定を試す：`SDL_VIDEODRIVER=x11`（Linux）
- macOS: システム設定 > プライバシーとセキュリティ > 画面収録でTerminal/Terminal.appへのアクセスを許可

### ユニットがクリックに反応しない

- ユニットスプライト**上**をクリックしていることを確認（その近くではない）
- ゲームが一時停止されていないことを確認（**Space**を押して一時停止を切り替え）
- ユニットが実際に選択されていることを確認（黄色いハイライトリングを探す）

### 音が聞こえない

- システムボリュームがミュートされていないことを確認
- PyCC2はオーディオを**手続き的**に生成します — 外部音声ファイルは不要またはロードされません
- 一部のCI/ヘッドレス環境ではオーディオハードウェアをサポートしていません — これは非致命的で、ゲームは静かに動作します
- テストする：`python -c "import pygame; pygame.init(); pygame.mixer.init(); print(pygame.mixer.get_init())"`

### ゲームが遅いまたはラグがある

- 設定メニュー（F10）でウィンドウサイズを縮小
- 低品質プリセットに変更（設定メニューF10）
- **ネイティブPython**で実行していることを確認（Apple Silicon MacでのRosetta翻訳ではない）
- HUDでFPSカウンターを確認 — 一貫して30以下の場合、マップサイズまたはユニット数の削減を検討

### セーブされたゲームがロードできない

- `PyCC2/saves/`ディレクトリが存在し`.json`ファイルを含むことを確認
- ステータスが**破損**を示す場合: ファイルが改ざんされています — 削除して新しいセーブを作成
- ステータスが**互換性なし**を示す場合: バージョン間でセーブ形式が変更されました — 古いセーブを削除、移行できません
- クイックセーブスロット（スロット0）はF5を押すたびに上書きされます — 名前付きセーブには他のスロットを使用

### AIが簡単すぎる/難しすぎる

- 難易度は設定メニュー（F10 → ゲームプレイタブ）で設定可能
- 5つのプリセット: RECRUIT / EASY / NORMAL / HARD / VETERAN
- 18個の個別パラメータは`DifficultySystem`構成で調整可能

## 用語集

| 用語 | 定義 |
|------|------|
| **弾道エンジン** | 距離減衰、掩体修正、および乱数分散を使用して弾道 trajectory、命中率、ダメージを計算 |
| **ビヘイビアツリー（BT）** | 各ユニットが戦術的選択（移動、射撃、リロード、撤退）を行うために使用する階層的AI意思決定構造 |
| **ブラックボード** | AIユニットが戦術情報（ターゲット割り当て、脅威レベル、巡回状態）を読み書きする共有メモリ構造 |
| **掩体ボーナス** | 特定の地形を占領時の敵命中率の割合減少（建物 > 森林 > 弹痕 > 開地） |
| **隠蔽修正** | 検出確率の減少 — 値が高い = 敵がこのユニットを発見するのがより困難 |
| **ドメイン駆動設計（DDD）** | ビジネスロジック（ドメイン層）を技術的懸念（プレゼンテーション/インフラ）から分離するアーキテクチャパターン |
| **EventBus | デカップルされたゲームコンポーネントを接続する pub/sub メッセージングシステム |
| **フォッグ・オブ・ウォー** | あなたのいずれのユニットにも見えていない領域が表示とAI知覚の両方から隠される |
| **HMAC-SHA256** | セーブファイルが改ざんされていないことを保証するハッシュベースのメッセージ認証コード |
| **LOS（視線） | 射手からターゲットまでの直線パスが遮断地形（森林、固体建物、壁、水域）によって妨げられていないかどうか |
| **士気** | ユニットの戦闘意志（0–100スケール）；低い士気は精度を低下させ、敗走を引き起こす可能性がある |
| **ピクセルアート** | 画像が pixel-by-pixel で描画されるグラフィックススタイル、オリジナルCC2の美学に一致 |
| **手続き的生成** | 外部 art/audio アセットファイルではなくアルゴリズムによってコンテンツ（スプライト、サウンド）を作成 |
| **ステートマシン** | ユニット lifecycle を管理する有限状態コントローラ：IDLE → MOVING → ATTACKING → RELOADING → DEAD |
| **制圧** | 重い火が士気を30未満に駆動し、有効性を大幅に低下させる戦闘状態 |
| **タイル** | ゲームマップグリッドの1平方単位; 実世界の距離約10メートルを表現 |
| **UPS（1秒あたりの更新数）** | ゲームロジック tick レート; PyCC2は固定30 UPSでロジックを実行し、ターゲットFPSでレンダリング補間 |
| **ユニットテンプレート** | ユニットタイプのすべてのステータスを含むデータ定義（HP、武器、速度、視界など） |
| **勝利条件** | 戦闘が終了し誰が勝つかを決定するルールのセット（司令官キル、排除、士気崩壊、目標、時間） |
