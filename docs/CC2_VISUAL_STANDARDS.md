## 🎨 1. 视觉风格总览

### 1.1 核心特征
- **视角**: 等距视角（Isometric），45度俯视
- **时代**: 二战（1944-1945）
- **风格**: 写实主义像素艺术
- **色调**: 自然色调，略带战争氛围的灰暗感

### 1.2 分辨率标准
- **原版分辨率**: 640x480 或 800x600
- **Tile尺寸**: 32x32 像素（等距投影后）
- **单位精灵**: 32x32 到 48x48 像素
- **UI元素**: 固定像素尺寸

## 🗺️ 2. 地形系统标准

### 2.1 地形类型
根据原版CC2，地形包括以下类型：

| ID | 类型 | 描述 | 视觉特征 |
|----|------|------|----------|
| 0 | 草地 (Grass) | 开阔地形 | 绿色，有纹理细节 |
| 1 | 道路 (Road) | 硬质路面 | 灰褐色，有车辙痕迹 |
| 2 | 森林 (Forest) | 密集树木 | 深绿色，树冠覆盖 |
| 3 | 建筑 (Building) | 房屋/谷仓 | 立体等距视角 |
| 4 | 墙壁 (Wall) | 石墙/围墙 | 灰色，有厚度感 |
| 5 | 水域 (Water) | 河流/池塘 | 蓝色，有波纹 |
| 6 | 桥梁 (Bridge) | 可通行桥 | 木质或石质 |
| 7 | 泥地 (Mud) | 松软地面 | 褐色，影响移动 |
| 8 | 沙地 (Sand) | 沙质地面 | 浅黄色 |
| 9 | 雪地 (Snow) | 积雪地面 | 白色，冬季场景 |
| 10 | 弹坑 (Crater) | 爆炸坑 | 深褐色凹陷 |
| 11 | 铁丝网 (Wire) | 障碍物 | 灰色网状 |
| 12 | 壕沟 (Trench) | 防御工事 | 褐色，有深度 |
| 13 | 掩体 (Bunker) | 混凝土工事 | 灰色，坚固 |

### 2.2 地形渲染标准
- **等距投影**: 所有地形tile使用等距视角
- **过渡平滑**: 不同地形类型之间有平滑过渡
- **细节纹理**: 每个tile有丰富的纹理细节（草地有草丛、道路有石子）
- **阴影效果**: 建筑物和树木投射阴影
- **高度差**: 通过视觉暗示表现地形高度

### 2.3 Terrain.azp 格式
- **文件格式**: 专有的.azp压缩格式
- **Tile数据**: 包含所有地形tile的像素数据
- **调色板**: 使用256色调色板
- **压缩**: 使用RLE或类似压缩算法

## 👥 3. 单位精灵标准

### 3.1 单位类型

#### 步兵单位
| 单位类型 | 尺寸 | 方向数 | 动画帧 | 特征 |
|---------|------|--------|--------|------|
| 步兵班 (Infantry Squad) | 32x32 | 8 | 4-6 | 多个士兵组成 |
| 机枪组 (MG Team) | 32x32 | 8 | 4 | 机枪+弹药手 |
| 指挥官 (Commander) | 32x32 | 8 | 4 | 军官制服 |
| 狙击手 (Sniper) | 32x32 | 8 | 4 | 单兵，伪装 |
| 工兵 (Engineer) | 32x32 | 8 | 4 | 携带工具 |
| 医疗兵 (Medic) | 32x32 | 8 | 4 | 红十字标识 |

#### 载具单位
| 单位类型 | 尺寸 | 方向数 | 动画帧 | 特征 |
|---------|------|--------|--------|------|
| 轻型坦克 | 48x48 | 8 | 2 | 小型炮塔 |
| 中型坦克 (Sherman/PzIV) | 48x48 | 8 | 2 | 标准坦克 |
| 重型坦克 (Tiger/IS-2) | 56x56 | 8 | 2 | 大型炮塔 |
| 半履带车 | 40x40 | 8 | 2 | 前轮+后履带 |
| 吉普车 | 32x32 | 8 | 2 | 轻型车辆 |
| 卡车 | 40x40 | 8 | 2 | 运输车辆 |

### 3.2 精灵渲染标准

#### 方向系统
- **8方向**: N, NE, E, SE, S, SW, W, NW
- **角度**: 每个方向45度
- **命名**: `unit_type_d0.spr` 到 `unit_type_d7.spr`

#### 动画系统
- **站立**: 1-2帧（呼吸动画）
- **移动**: 4-6帧（行走循环）
- **射击**: 2-4帧（开火+后坐力）
- **死亡**: 3-5帧（倒下动画）

#### 视觉细节
- **等距视角**: 45度俯视，保持透视一致
- **阴影**: 单位下方有椭圆阴影
- **细节**: 可识别武器、装备、制服
- **派系区分**: 
  - 盟军: 橄榄绿/卡其色
  - 德军: 灰绿色/灰色

### 3.3 SPRI/IRPS 文件格式
- **文件扩展名**: .spr
- **结构**: 头部 + 调色板 + 像素数据
- **透明色**: 使用特定颜色索引表示透明
- **压缩**: 可能使用RLE压缩

## 🏗️ 4. 建筑物标准

### 4.1 建筑类型
| 建筑类型 | 尺寸 | 可进入 | 视觉特征 |
|---------|------|--------|----------|
| 民房 (House) | 64x64 | 是 | 2层，窗户，门 |
| 谷仓 (Barn) | 64x80 | 是 | 大型，木质 |
| 教堂 (Church) | 80x96 | 是 | 尖顶，十字架 |
| 工厂 (Factory) | 96x96 | 是 | 烟囱，大窗 |
| 掩体 (Bunker) | 48x48 | 是 | 混凝土，射击孔 |
| 围墙 (Wall) | 32x16 | 否 | 石墙段 |

### 4.2 建筑渲染标准
- **等距3D**: 建筑物有明显的立体感
- **多层次**: 屋顶、墙体、基础分层
- **破损状态**: 3-5个破损等级（完好→轻伤→重伤→废墟）
- **窗户**: 可见窗户，被占领时显示士兵
- **阴影**: 建筑物投射阴影到地面

## 🎮 5. UI元素标准

### 5.1 主界面布局
```
┌─────────────────────────────────────────────────────────┐
│  [菜单栏]                                    [时间/天气] │
├──────────┬──────────────────────────────────┬───────────┤
│          │                                  │           │
│  单位    │                                  │  小地图   │
│  信息    │        主游戏视图                │           │
│  面板    │      (等距战场)                  │  (战术)   │
│          │                                  │           │
│  [头像]  │                                  │  [标记]   │
│  [状态]  │                                  │           │
│  [武器]  │                                  │           │
├──────────┴──────────────────────────────────┴───────────┤
│  [命令栏: 移动|攻击|防御|撤退|烟雾|...]                  │
└─────────────────────────────────────────────────────────┘
```

### 5.2 UI组件标准

#### 单位信息面板
- **位置**: 左侧
- **尺寸**: 约200x400像素
- **内容**:
  - 单位头像/图标
  - 单位名称和类型
  - 士气条（绿→黄→红）
  - 弹药数量
  - 人员数量
  - 状态图标（疲劳、压制、受伤）

#### 小地图
- **位置**: 右下角
- **尺寸**: 约150x150像素
- **特征**:
  - 战场全景
  - 单位标记（蓝色=盟军，红色=德军）
  - 地形简化显示
  - 视野范围指示

#### 命令栏
- **位置**: 底部
- **尺寸**: 全宽，约60像素高
- **按钮**: 
  - 图标式按钮（32x32）
  - 鼠标悬停提示
  - 快捷键显示

## ✨ 6. 视觉效果标准

### 6.1 战斗效果

#### 爆炸效果
- **类型**: 手榴弹、炮弹、坦克炮
- **动画帧**: 6-8帧
- **尺寸**: 32x32 到 64x64
- **特征**: 火光→烟雾→消散
- **颜色**: 橙红色火光，灰黑色烟雾

#### 枪口火光
- **动画帧**: 2-3帧
- **尺寸**: 16x16
- **特征**: 黄白色闪光

#### 烟雾效果
- **类型**: 烟雾弹、燃烧、尘土
- **动画帧**: 8-12帧（循环）
- **尺寸**: 48x48 到 96x96
- **特征**: 半透明，逐渐扩散

#### 弹道轨迹
- **类型**: 子弹、炮弹
- **渲染**: 线条或粒子
- **颜色**: 黄色（曳光弹）

### 6.2 环境效果

#### 天气效果
| 天气 | 视觉效果 | 影响 |
|------|---------|------|
| 晴天 | 明亮，清晰阴影 | 视野最佳 |
| 阴天 | 柔和光照，弱阴影 | 视野正常 |
| 雨天 | 雨滴粒子，地面反光 | 视野降低 |
| 雾天 | 白色雾气，视野受限 | 视野大幅降低 |
| 雪天 | 雪花粒子，白色地面 | 视野降低，移动慢 |

#### 时间效果
- **黎明**: 橙红色调
- **白天**: 自然光照
- **黄昏**: 金黄色调
- **夜晚**: 蓝黑色调，月光

## 📊 7. 性能标准

### 7.1 帧率目标
- **最低**: 30 FPS
- **推荐**: 60 FPS
- **原版**: 约30-40 FPS（1990年代硬件）

### 7.2 资源占用
- **内存**: 原版约32-64MB
- **显存**: 原版约4-8MB
- **现代目标**: <500MB内存，<200MB显存

### 7.3 优化策略
- **精灵缓存**: 预加载常用精灵
- **视野裁剪**: 只渲染可见区域
- **LOD系统**: 远距离简化渲染
- **批量渲染**: 合并相同类型的绘制调用
