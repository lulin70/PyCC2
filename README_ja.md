## 概要

**PyCC2**は、**Close Combat 2: A Bridge Too Far**（Atomic Games, 1997）のスクラッチ再実装です。史上最も影響力のあるリアルタイム戦術ウォーゲームの一つです。1944年9月の**マーケット・ガーデン作戦（Operation Market Garden）**を舞台に、オランダの田園地帯で枢軸軍との過酷な接近戦において連合軍歩兵部隊を指揮します。

完全に**Python**と**pygame**で構築され、手続き的に生成されたピクセルアートスプライト、ゼロの音声ファイル依存性、およびビヘイビアツリーと司令官レベルの意思決定を備えた洗練されたAI対戦相手を特徴としています。

> [!NOTE]
> これは**ベータ候補品質のリメイク**です — コア戦闘、視覚、およびオーディオシステムが完成した完全にプレイ可能な状態です。詳細については[プロジェクトステータス](#project-status)をご覧ください。

## 機能

### コアゲームプレイ
| 機能 | 説明 |
|------|------|
| **リアルタイム戦術戦闘** | 部隊をリアルタイムで指揮；ターン制なし、一時停止なし（自分以外） |
| **弾道解決 | 物理ベースの弾道計算：距離減衰、掩体貫通、士気効果 |
| **8種類のユニット** | 歩兵、機関銃、対戦車砲、迫撃砲、**戦車**、**スナイパー**、衛生兵、司令官 |
| **14種類の地形 | 森林、建物、水域、弹痕、沼地、橋、生け垣 — 各々が移動と戦闘に影響 |
| **勝利条件 | 司令官の排除 / 全敵の破壊 / 声潰倒 / 目標の確保 / 時間制限 |

### 人工知能
- **ビヘイビアツリーシステム** — 各AIユニットが独立した戦術的決定を行う
- **司令官AI** — 戦場全体の評価、脅威評価、命令生成
- **部隊調整** — 火力集中、交差支援、側面攻撃戦術
- **4つの難易度レベル** — 新兵 → 通常 → ベテラン → 英雄（18個の調整可能パラメータ）
- **知覚システム** — 動的な視線（LOS）、フォッグ・オブ・ウォー、スナイパーのステルスボーナス

### 視覚とオーディオ
| システム | 詳細 |
|----------|------|
| **ピクセルアートエンジン** | 手続き的スプライト生成、28色の軍事パレット（連合軍オリーブドラブ vs 枢軸軍フェルトグラウ） |
| **ユニットアニメーション** | ユニットあたり5状態：待機、移動、攻撃、リロード、死亡 |
| **パーティクルエフェクト** | マズルフラッシュ、血液飛散、煙、残骸、火花、薬莢、爆発リング、土煙 |
| **画面振動 | 衝撃応答型カメラ振動と指数減衰 |
| **ダメージ数値 | グレード化された色/スケール：致命的なキルは赤、高ダメージは橙、低ダメージは黄 |
| **手続き的オーディオ** | 数学的に生成された音声タイプ — ゼロの外部音声ファイル不要 |
| **HDディスプレイ** | 動的タイルサイズ、Retina DPI自動検出、1440+解像度サポート |

### システム
- **安全なセーブ/ロード** — HMAC-SHA256署名付きJSONセーブ、8スロット、改ざん検出
- **キャンペーンモード** — **5つのミッション定義**（チュートリアル → 橋梁強襲 → 防衛線維持 → 反撃 → 総攻撃）
- **チュートリアルシステム** — 新規プレイヤー向けの対話型ガイド
- **パフォーマンス最適化** — GameLoop分解（4コンポーネント）、改善されたフレームタイミング
- **設定メニュー** — ゲーム内設定（4タブ：グラフィックス、オーディオ、ゲームプレイ、コントロール）
- **戦闘統計** | キル/デストラッキング、命中率パーセンテージ、与ダメージ、プレイ時間 |
- **戦後パネル** | 勝利/敗北画面と完全な統計情報 |

## インストール

### 前提条件

- **Python 3.11+** (`match/case`構文、タイプパラメータ構文を使用; 3.12+推奨)
- **macOS / Linux / Windows** (pygameクロスプラットフォーム)
- **約200MBのディスク容量** (主にPythonパッケージ用)

### クイックインストール

```bash
# リポジトリをクローン
git clone https://github.com/lulin70/PyCC2.git
cd PyCC2

# 仮想環境を作成（推奨）
python -m venv .venv
source .venv/bin/activate   # macOS/Linux
# .venv\Scripts\activate    # Windows

# 編集可能モードでインストール
pip install -e .

# インストールを確認
python -m pytest tests/ -q  # ~3985 passed と表示されるはず
```

詳細なインストール手順については [INSTALL_ja.md](INSTALL_ja.md) をご覧ください。

### 操作方法

| 入力 | アクション |
|------|-----------|
| **左クリック** | ユニットを選択 |
| **Shift + 左クリック** | 複数ユニットを選択 |
| **右クリック（地面）** | 選択ユニットを移動 |
| **右クリック（敵）** | ターゲットを攻撃 |
| **M** | 移動モードに切り替え |
| **A** | 攻撃モードに切り替え |
| **S** | 現在のアクションを停止 |
| **K** | 発煙弾（計画中） |
| **D** | 塹構を掘る / 掩体を取る |
| **ESC** | 選択解除 / 一時停止 |
| **W/A/S/D** | カメラ移動 |
| **マウスホイール** | ズームイン/アウト |
| **F11** | フルスクリーンモード切り替え |
| **F1** | ヘルプ / チュートリアルオーバーレイ |
| **F3** | デバッグ情報オーバーレイ |
| **F5** | クイックセーブ |
| **F9** | クイックロード |

## テスト

```bash
# 完全なテストスイートを実行（約~3985テスト、約32秒）
pytest tests/ -q

# 特定のテストカテゴリを実行
pytest tests/unit/ -q                    # ユニットテストのみ
pytest tests/integration/ -q             # 結合テスト
pytest tests/e2e/ -q                     # エンドツーエンドテスト

# カバレッジレポート付き
pytest tests/ --cov=src/pycc2 --cov-report=html
```

**カバレッジ**: ユニット/結合/E2Eカテゴリ全体で~3985テスト、すべてのドメインロジック、システム統合、視覚レンダリングパイプラインを対象。
